## 🔹 Lista de Compras - Gerenciador de Itens

Aplicação funcional desenvolvida para auxiliar na organização de tarefas domésticas, focada na manipulação de listas dinâmicas e persistência de dados no navegador.

## 🚀 Habilidades aplicadas

* **Persistência com LocalStorage:** Implementação de lógica para salvar, ler e excluir dados localmente, garantindo que as informações permaneçam salvas após o fechamento do navegador.
* **Manipulação Dinâmica de Listas:** Criação, edição e remoção de elementos do DOM em tempo real utilizando JavaScript Vanilla.
* **Validação de Formulários:** Tratamento de entradas do usuário para evitar itens vazios ou duplicados, melhorando a integridade dos dados.
* **Event Listening:** Uso eficiente de ouvintes de eventos para interações como marcar itens como "concluídos" ou excluir da lista.
* **UX e Feedback Visual:** Estilização condicional para diferenciar itens pendentes de itens comprados.

Este projeto demonstra minha transição de carreira ao aplicar o pensamento estruturado de processos internos na criação de uma ferramenta utilitária robusta e intuitiva.
